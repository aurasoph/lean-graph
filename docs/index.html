🛠️ CLI Tool Usage

Generate custom dependency graphs for your own Lean projects:

# Install the tool lake run MainGraph --help # Generate different graph types lake run MainGraph imports lake run MainGraph proof-deps lake run MainGraph type-deps lake run MainGraph structures # Output to specific file lake run MainGraph imports --to my_graph.dot

🌐 Interactive Web Explorer

For local exploration with search and traversal features:

cd web-explorer python -m http.server 8000 # Visit localhost:8000

📁 DOT File Viewing

View the downloaded .dot files with:

# Using Graphviz dot -Tsvg graph.dot -o graph.svg # Using online viewers # Upload to: https://dreampuf.github.io/GraphvizOnline/

📚 About This Project

This tool analyzes dependency relationships in Lean 4 mathematical libraries, with a focus on Mathlib. It extracts and visualizes different types of dependencies to help researchers understand the structure of mathematical formalization.

The graphs above represent the current state of Mathlib (~April 2024) and showcase the intricate relationships between mathematical concepts, proofs, and their formal representations.
